# Weather Balloon Simulation Telemetry Web Map

A interactive web map that visualizes simulated weather balloon telemetry data, including timestamps, altitude, latitude, longitutde, tempurature, and pressure.

The goal of the project is to transform raw telemetry data into a visual, geographic, interactive tool for analyzing weather balloon behavior.

## Project Overview:

This project consists of two main components:
  1. **Weather Balloon Simulation (Java Program):**
     A Java based [simulation](WeatherBalloonSimulation.java) that generates semi-realistic telemetry data and is created into a CSV file.
  2. **Interactive Web Map Visualization:**
     The generated CSV data file is used with QGIS and Google Satellite mapping to create an interactive geographic visualization of a weather balloon's simulated flight path.
A sample dataset generated by the Java Simulation is included to currently demonstrate the [web map](https://o-flores1.github.io/GIS-Weather-Balloon-Telemetry-map/) interactivity.

## Simulated Telemetry Data
The Java Simulation generateds the following data:
 - Latitude
 - Longitude
 - Altitude
 - Tempurature
 - Pressure
 - Timestamp

 ## Web Map Feathers
  - Plots telemetry data points in 60 second intervals on a geographic map.
  - Hover functionality to display corresponding data values.
  - Visual representation of balloons position over time.
  - Zoom controls
